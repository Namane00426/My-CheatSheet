Cheat sheet: Git&GitHub Commands Reference

Git Commands

Command Description
pwd view now's location
rm -r -directory name- Remove directory
cp -filA- -fileB copy fileA to fileB
mv fileA -directoryB- Move fileA to B
cd ~ Move to home directory
mkdir -directory name- Make a new directory
touch -file name- Make a new file

GitHub commands

Commands Description
git branch -m -new branch name- Change to new branch
rm -rf .git Remove git
git reset --hard -commit code- Reset commited file
git remote add origin -url- Add remote repository
git rmote rm origin -url- Remove remote repository commited file
git remote -v View remote repository
git merge --allow-unrelated-histories origin/main merge remote repo (Use it, when push is rejected!)
